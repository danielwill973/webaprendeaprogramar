¿Qué es Python?

Python es un lenguaje de programación interpretado, de alto nivel y multipropósito, creado por Guido van Rossum en 1991. Es uno de los lenguajes más populares actualmente gracias a su sintaxis clara y versatilidad.

Características principales:

  • Sintaxis legible: Similar al inglés, con indentación obligatoria
  • Multiplataforma: Funciona en Windows, macOS y Linux
  • Tipado dinámico: No necesitas declarar tipos de variables
  • Gran ecosistema: Más de 300,000 librerías disponibles
  • Comunidad activa: Una de las comunidades más grandes del mundo

¿Para qué se usa Python?

Desarrollo Web

Con frameworks como Django y Flask

Inteligencia Artificial

Librerías como TensorFlow y PyTorch

Análisis de Datos

Pandas, NumPy y Matplotlib

Automatización

Scripts para automatizar tareas

Instalación de Python

Para comenzar a programar en Python, sigue estos pasos:

Windows/macOS

  1. Descarga la última versión de Python desde python.org
  2. Ejecuta el instalador
  3. Marca la opción "Add Python to PATH" (importante)
  4. Haz clic en "Install Now"
  5. Verifica la instalación abriendo la terminal y escribiendo python --version

Linux (Ubuntu/Debian)

# Actualizar paquetes
sudo apt update

# Instalar Python 3
sudo apt install python3

# Verificar versión
python3 --version

Sintaxis Básica

Python se caracteriza por su sintaxis clara y legible. Aquí los fundamentos:

Hola Mundo

# Esto es un comentario
print("¡Hola Mundo!")  # Imprime texto en consola

Indentación

Python usa indentación (espacios/tabs) para definir bloques de código:

if 5 > 2:
    print("Cinco es mayor que dos")  # 4 espacios de indentación
    print("Esto está dentro del if")
    
print("Esto está fuera del if")

Comentarios

# Esto es un comentario de una línea

"""
Este es un comentario
multilínea (docstring)
"""

print("Hola")  # Comentario después de código

Variables y Tipos de Datos

En Python no necesitas declarar el tipo de variable:

# Ejemplos de variables
nombre = "María"     # String (texto)
edad = 28            # Entero (int)
altura = 1.68        # Decimal (float)
es_estudiante = True # Booleano (True/False)

Operaciones básicas

a = 10
b = 3

suma = a + b       # 13
resta = a - b      # 7
multiplicacion = a * b  # 30
division = a / b   # 3.333...
division_entera = a // b  # 3
modulo = a % b     # 1 (resto de división)
potencia = a ** b  # 1000

Conversión de tipos

# Convertir a string
numero = 42
texto = str(numero)  # "42"

# Convertir a entero
texto = "123"
numero = int(texto)  # 123

# Convertir a float
texto = "3.14"
decimal = float(texto)  # 3.14

Condicionales

Estructuras para controlar el flujo del programa:

if simple

edad = 18
if edad >= 18:
    print("Eres mayor de edad")

if-else

temperatura = 25
if temperatura > 30:
    print("Hace calor")
else:
    print("Temperatura normal")

if-elif-else

puntaje = 85
if puntaje >= 90:
    print("Excelente")
elif puntaje >= 70:
    print("Bueno")
else:
    print("Necesitas mejorar")

Operadores de comparación

Operador Significado Ejemplo
== Igual a 5 == 5 → True
!= Diferente de 5 != 3 → True
> Mayor que 5 > 3 → True
< Menor que 5 < 3 → False
>= Mayor o igual 5 >= 5 → True
<= Menor o igual 5 <= 3 → False

Ejercicios Prácticos

Practica lo aprendido con estos ejercicios:

Ejercicio 1: Calculadora básica

Crea un programa que pida dos números al usuario y muestre:

  • La suma
  • La resta
  • La multiplicación
  • La división
num1 = float(input("Ingresa el primer número: "))
num2 = float(input("Ingresa el segundo número: "))

print(f"Suma: {num1 + num2}")
print(f"Resta: {num1 - num2}")
print(f"Multiplicación: {num1 * num2}")
print(f"División: {num1 / num2}")

Ejercicio 2: Verificador de edad

Crea un programa que:

  • Pida la edad al usuario
  • Si es mayor o igual a 18, mostrar "Puedes votar"
  • Si es mayor o igual a 16, mostrar "Puedes votar en algunas elecciones"
  • Si es menor, mostrar "No puedes votar"
edad = int(input("Ingresa tu edad: "))

if edad >= 18:
    print("Puedes votar")
elif edad >= 16:
    print("Puedes votar en algunas elecciones")
else:
    print("No puedes votar")

Ejercicio 3: Clasificador de temperatura

Crea un programa que clasifique la temperatura en:

  • Frío: menos de 10°C
  • Templado: 10-24°C
  • Calor: 25°C o más
temperatura = float(input("Ingresa la temperatura: "))

if temperatura < 10:
    print("Hace frío")
elif 10 <= temperatura <= 24:
    print("Temperatura templada")
else:
    print("Hace calor")
